# StoryTailor.ai 프로젝트 로드맵

이 문서는 StoryTailor.ai 프로젝트의 개발 로드맵을 설명합니다. 각 단계(Phase)별 목표와 주요 마일스톤을 확인할 수 있습니다.

## 🎯 비전

> **"모든 어린이에게 안전하고 개인화된 AI 스토리텔링 경험을 제공한다"**

StoryTailor.ai는 생성형 AI와 머신러닝을 활용하여 아이들의 읽기 수준, 선호도, 학습 목표에 맞춘 맞춤형 이야기를 제공하는 것을 목표로 합니다.

---

```
2026 Q1          2026 Q2          2026 Q3          2026 Q4
   │                │                │                │
   ├─ Phase 1 ─────►├─ Phase 2 ─────►├─ Phase 3 ─────►├─ Phase 4
   │  핵심 안정화    │  음성 기능      │  추천/분석      │  완성/출시

```

## 🚀 Phase 1: 핵심 기능 안정화 (2026 Q1)

**목표**: 핵심 스토리 생성 기능의 안정성 확보 및 테스트 강화

### 마일스톤

- [x] 핵심 스토리 생성 엔진 개발
- [x] RAG 기반 환각 방지 시스템 구축
- [x] FastAPI 서버 구축
- [x] 안전 필터링 모듈 개발
- [ ] 단위 테스트 커버리지 80% 달성
- [ ] 통합 테스트 자동화
- [ ] API 문서화 완료 (Swagger/OpenAPI)
- [ ] 성능 최적화 및 부하 테스트

### 주요 결과물
- 안정적인 스토리 생성 API
- 사실 기반 콘텐츠 생성 (RAG)
- 아동 안전 콘텐츠 필터링

---

## 🎙️ Phase 2: 음성 기능 및 읽기 진단 (2026 Q2)

**목표**: 음성 기반 읽기 기능과 읽기 수준 진단 시스템 개발

### 마일스톤

- [ ] Text-to-Speech (TTS) 통합
  - [ ] 다양한 음성 옵션 (남성/여성, 속도 조절)
  - [ ] 감정 표현이 담긴 음성 생성
- [ ] Speech-to-Text (STT) 통합
  - [ ] 아이 음성 인식 최적화
  - [ ] 발음 정확도 피드백
- [ ] 읽기 수준 진단 시스템
  - [ ] Lexile 기반 평가 알고리즘
  - [ ] 읽기 유창성 측정
  - [ ] 어휘력 평가

### 주요 결과물
- 음성으로 이야기 들려주기 기능
- 아이 음성 읽기 연습 지원
- 자동 읽기 능력 평가 시스템
